# Thunder Wing

An original vertical shooter built for the Huangshan Pi. Keep the board level
during the short launch calibration, then tilt left or right to steer. Larger
tilt angles produce higher horizontal speed.

The fighter fires automatically. Destroy enemy waves, collect power cores for
spread fire, avoid enemy bullets, and defeat the stage boss. Touch the pause
button during a sortie or use RETRY after losing all three lives.
